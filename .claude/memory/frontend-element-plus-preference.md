## 编码偏好规则

前端编码优先使用 Element Plus 组件。

**Why:** 项目已集成 Element Plus，需要保持 UI 风格一致性。

**How to apply:** 在创建表单、弹窗、按钮、输入框等 UI 组件时，优先使用 Element Plus 提供的组件（el-form、el-input、el-button、el-dialog 等），而非原生 HTML 元素。
