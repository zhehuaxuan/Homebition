# 关于我页面菜单优化 - 需求文档

- **REQ-ID**: REQ-027
- **日期**: 2026-07-20
- **模块**: 前端 - 关于我页面

## 背景

当前"关于我"页面侧边栏菜单项过多（7 项），且功能混杂——个人简介、设备、文章管理、任务管理、标签管理、订阅管理、每日总结全部平铺在同一个列表中，导航效率低，视觉效果杂乱。

## 目标

对"关于我"页面的侧边栏菜单进行分组优化，提升导航清晰度和使用体验。

## 分组方案

将 7 个菜单项按功能划分为 3 组：

| 分组 | 菜单项 |
|------|--------|
| **个人** | 关于我、我的设备 |
| **内容管理** | 文章清单、标签管理 |
| **任务与订阅** | 任务清单、订阅管理、每日总结 |

## UI 设计

### PC 端（≥768px）— 侧边栏分组

```
┌─────────────────┐
│  关于我          │  ← sidebar-header
├─────────────────┤
│  个人            │  ← group-title
│    👤 关于我      │
│    💻 我的设备    │
├─────────────────┤
│  内容管理         │  ← group-title
│    📄 文章清单    │
│    🏷 标签管理    │
├─────────────────┤
│  任务与订阅       │  ← group-title
│    📋 任务清单    │
│    📧 订阅管理    │
│    📋 每日总结    │
└─────────────────┘
```

**样式细节：**
- 分组标题（group-title）：字号 12px，颜色 #64748b，上下留 8px 边距，左右留 20px
- 分组之间用淡色分割线（border-color: #334155，与 sidebar-header 底部一致）
- 菜单项样式保持现有 hover 和 active 高亮不变
- 图标差异化：标签管理从 📄 改为 🏷

### 移动端（<768px）— 分段切换 + 标签栏

```
┌─────── 个人 ───────┬─ 内容管理 ─┬─ 任务与订阅 ─┐  ← 分段控制器
├──────── 可左右滑动 ─────────────────┤
│  👤 关于我   │  💻 我的设备       │  ← 仅显示当前分组下的菜单
└────────────────────────────────────┘
↓
[router-view 内容区]
```

**交互说明：**
- 顶部使用分段控制器切换分组
- 下方为可滚动的标签栏，只显示当前选中分组下的子菜单项
- 点击菜单项切换 router-view 内容
- 当前选中的菜单项高亮

## 涉及修改的文件

| 文件 | 改动 |
|------|------|
| `client/src/views/About.vue` | 侧边栏菜单改为分组结构，移动端改为分段控制器 + 动态标签栏 |

## 不涉及修改的部分

- 路由配置（router/index.js）不变
- 子页面组件功能不变
- 后端不变
